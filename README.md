



# 📧 Auto Email Sender Pro

A Python tool to send emails automatically via Gmail.

## Features
- 📩 Send to multiple recipients
- 📎 File attachment support
- 📜 Email history saved in history.txt
- 🔒 Secure App Password login
- ⚠️ Error handling

## How to Run
```bash
python main.py
```

## Gmail Setup
1. Enable 2-Step Verification
2. Generate App Password
3. Use that password in program

## Technologies Used
- Python
- smtplib
- email.mime
