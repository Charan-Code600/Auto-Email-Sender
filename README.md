


# Auto Email Sender 📧

A simple Python tool to send emails automatically via Gmail.

## Features
- ✅ Send emails via Gmail
- ✅ Custom subject and message
- ✅ Secure TLS connection
- ✅ Error handling

## Requirements
No extra installation needed!
Python built-in libraries used.

## Setup
1. Go to Google Account Settings
2. Security → 2-Step Verification → On
3. App Passwords → Generate password
4. Use that password in the program

## How to Use
1. Run the program:
   python auto_email_sender.py
2. Enter your Gmail address
3. Enter App Password
4. Enter receiver email
5. Write subject and message
6. Email sent! ✅

## Important Note
⚠️ Never share your App Password with anyone!
